QLALR Adventures Example
========================

This package contains the source code for the example described in the
"QLALR Adventures" article in Qt Quarterly 33.

One example is provided. This can be built using the qmake project file
supplied.

About Qt Quarterly
==================

Qt Quarterly is a newsletter available to Qt developers. Every quarter we
